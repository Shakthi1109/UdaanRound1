
Instructions:

1. Install python3 and flask in system
2. Extract the folder in the directory where python3 and flask are installed
3. Open Command prompt and go to the housekeeping folder's directory
4. Run flask:
	C:\......\housekeeping>set FLASK_APP=index.py
	C:\......\housekeeping>set FLASK_ENV=development
	C:\......\housekeeping>flask run
5. Run URLs in browser: 
	http://127.0.0.1:5000/add-asset
	http://127.0.0.1:5000/add-task
	http://127.0.0.1:5000/add-worker
	http://127.0.0.1:5000/asset/all
	http://127.0.0.1:5000/allocate-task
	http://127.0.0.1:5000/get-tasks-for-worker/<workerid>
